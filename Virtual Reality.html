Mixed Reality

The project showcase include both virtual reality and augmented realty projects that I have worked on. The projects use a variety of different technologies and platforms in which I had the opportunity to practice my skills and technical ability while trying new things.

AR Fish Tank

Using Reynold's Boids algorithm to simulate fish swimming in a 3D space designed in Maya. The focus is on the 3D environment and flocks which can be brought to life in the real world using vuforia and unity.

Calm VR Driving Loop

The inspiration for this came from the lo-fi music videos that show 3D environments with relaxing music. The goal was to create a similar experience in VR. The project was created using Unity and the Oculus Rift.

  • Learn more

VR Escape Room

Short prototype of an escape room in VR. There are some comedy-horror elements instilled in the scene however the focus in mainly on the mechanics of the game. The project was created using Unity and the Oculus Rift.

  • Learn more

Bringing Full-Body Haptics To VR

A small fun side project that uses the T.E.N.S unit to simulate the sensation of getting hit in VR. Please only replicate this project after consulting with a medical professional about the risks of T.E.N.S and if you are comfortable with the risks of using it.
